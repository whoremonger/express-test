Backend step guide

  1. Create project folder
  2. npm init to create node package.json file to list and keep track of installs and dependencies
  3. install express a backend framework
  4. create a server or index js file to run the dev server
  5. Create a .env file for to store environment variables for security. Use "process.env" to access it.
  6. Create a pages folder to store webpages and set the folder to a static folder for access
  7. create any homepage or css files and link it as well as bootstrap and such
  8. Install a template engine like EJS, handlebars, or pug
  9. create a router folder/file for complex sites
  10. create any models or data for prototyping (like users: name, email, etc)
  11. Install moment() for formatting dates
  12. Start to create api and use postman to test routes, data, and CRUD
  13. CRUD: Read(get), Create(post), Update(Put), Delete(destroy)
  14. Get req all is get '/route/item'
  15. Get req for single item is get '/route/:id'
  16. create middleware for req and res functions
  17. Common server error codes: 400 bad request, 401 unauthorized, 403, forbidden, 404 not found, 500 internal server error, 502 bad gateway, 503 service unavailable, 504 gateway timeout, 200 OK, 201 created, 202 accepted, 204 no content, 301 moved permanently, 304 not modified
  18. use res.status() for validating codes
  19. use req.body for form fields/json data and req.params for url value like id
  20. Start to validate user data / or using try catch to catch any errors
  21. Use Postman app to test routes, data, and error codes
  22. Link express with a database and set connection strings
  23. create/validate any JWT tokens for security.
  24. register user and session APIs for a User for a login/logout session
